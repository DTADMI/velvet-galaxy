# Brand Copy Velvet Docs

Load these docs when they match the copy surface:

- `components/velvet-logo.tsx`: logo display text and branding.
- `app/layout.tsx`: root layout metadata and title.
- `app/globals.css`: brand palette tokens.
- `lib/i18n.ts` and `lib/i18n/`: localization strings.
- `components/theme-provider.tsx`: theme storage keys and branding.
- `public/manifest.json`: PWA manifest branding.
